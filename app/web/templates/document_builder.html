Генератор документов

Новый генератор работает как юридическое интервью: определяет тип документа, собирает обязательные данные, показывает только недостающие вопросы и формирует Word по шаблону.

Выбранный клиент используется только как источник данных. LexPilot не считает автоматически, что клиент является истцом, заявителем или отправителем документа.
Следующий шаг Уточните недостающие данные

Заполните только те поля, которые известны. После повторного нажатия LexPilot обновит данные без повторного полного анализа запроса.

Финальный документ формируется по Word-шаблону. Проверка и правки остаются за юристом.
