# Универсальное исковое заявление

## Назначение

Основной шаблон искового заявления для LexPilot.

Используется в случаях, когда отсутствует специализированный шаблон иска.

Допускается использование для:

- взыскания задолженности;
- взыскания убытков;
- взыскания неустойки;
- расторжения договора;
- защиты прав потребителей;
- семейных споров;
- трудовых споров;
- иных гражданско-правовых требований.

---

## Обязательные данные

- суд;
- истец;
- ответчик;
- цена иска;
- обстоятельства спора;
- доказательства;
- правовое обоснование;
- исковые требования;
- приложения.
