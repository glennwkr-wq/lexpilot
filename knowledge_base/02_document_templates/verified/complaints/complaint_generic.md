&#x20;Общий шаблон жалобы



&#x20;Назначение



Используется для подготовки жалоб в государственные органы, органы местного самоуправления, надзорные органы, правоохранительные органы, должностным лицам и иным адресатам.



Является базовым шаблоном для всех видов жалоб в системе LexPilot.



\---



&#x20;Когда использовать



\- нарушение прав гражданина;

\- нарушение прав организации;

\- бездействие должностного лица;

\- незаконные действия должностного лица;

\- нарушение требований законодательства;

\- необходимость проведения проверки;

\- необходимость привлечения виновных лиц к ответственности.



\---



&#x20;Когда не использовать



\- для предъявления искового заявления;

\- для подачи апелляционной жалобы;

\- для подачи кассационной жалобы;

\- для направления претензии контрагенту;

\- для подачи ходатайства по судебному делу.
