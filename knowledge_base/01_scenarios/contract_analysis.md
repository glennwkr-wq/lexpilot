# Сценарий: анализ договора

Цель: выявить юридические риски в договоре.

Шаги:
1. Определить тип договора.
2. Выделить стороны, предмет, цену, сроки, ответственность.
3. Проверить существенные условия.
4. Найти рискованные формулировки.
5. Составить таблицу: риск — почему опасно — как исправить.
6. Отдельно указать, какие пункты требуют проверки юристом.
