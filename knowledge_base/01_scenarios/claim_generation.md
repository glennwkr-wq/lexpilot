# Сценарий: подготовка претензии

Цель: помочь юристу подготовить черновик претензии.

Шаги:
1. Определить тип спора.
2. Извлечь стороны, договор, сумму, сроки, нарушения.
3. Найти подходящий шаблон претензии.
4. Найти релевантные нормы и практику в базе знаний.
5. Сформировать структуру претензии.
6. Добавить блок требований.
7. Добавить список приложений.
8. Добавить блок: что проверить юристу.
