# Карта обязательных данных для отзыва на иск

## Обязательные данные

- суд;
- номер дела;
- истец;
- ответчик;
- требования истца;
- позиция ответчика;
- фактические возражения;
- правовые возражения;
- доказательства ответчика;
- просьба к суду;
- приложения.

## Дополнительные данные

- какие требования признаются;
- какие требования оспариваются;
- пропуск срока исковой давности;
- отсутствие доказательств у истца;
- неправильный расчет;
- судебная практика.

## Минимальный результат

LexPilot готовит отзыв по `response_generic.md`.
