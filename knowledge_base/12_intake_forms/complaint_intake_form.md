# Карта обязательных данных для жалобы

## Обязательные данные

- адресат жалобы;
- заявитель;
- лицо или орган, действия которого обжалуются;
- дата события;
- описание нарушения;
- нарушенные права;
- просьба заявителя;
- доказательства;
- приложения.

## Дополнительные данные

- нормы права;
- предыдущие обращения;
- ответ органа, если был;
- срок, в который нужно дать ответ;
- способ направления ответа.

## Минимальный результат

LexPilot готовит жалобу по `complaint_generic.md`.
