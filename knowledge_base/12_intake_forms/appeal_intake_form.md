# Карта обязательных данных для апелляционной жалобы

## Обязательные данные

- суд апелляционной инстанции;
- суд первой инстанции;
- номер дела;
- заявитель;
- процессуальный статус;
- дата решения;
- суть решения;
- в какой части решение обжалуется;
- основания несогласия;
- требования заявителя;
- приложения.

## Основания для проверки

- неправильное установление обстоятельств;
- неправильная оценка доказательств;
- нарушение норм материального права;
- нарушение норм процессуального права.

## Минимальный результат

LexPilot готовит апелляционную жалобу по `appeal_generic.md`.
