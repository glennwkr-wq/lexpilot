# Карта обязательных данных для искового заявления

## Цель

Перед подготовкой искового заявления LexPilot должен проверить наличие процессуальных и фактических данных.

## Обязательные данные

- суд;
- истец;
- ответчик;
- предмет спора;
- цена иска;
- обстоятельства дела;
- нарушенное право;
- доказательства;
- правовое обоснование;
- исковые требования;
- приложения.

## Дополнительные данные

- сведения о досудебном порядке;
- дата и номер договора;
- расчет взыскиваемой суммы;
- сведения о госпошлине;
- данные представителя;
- доверенность;
- судебная практика.

## Если данных не хватает

LexPilot должен сначала запросить недостающие сведения.

## Минимальный результат

Если данных достаточно, LexPilot готовит черновик по `lawsuit_generic.md`.
