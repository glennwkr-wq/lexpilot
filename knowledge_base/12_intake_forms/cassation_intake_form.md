# Карта обязательных данных для кассационной жалобы

## Обязательные данные

- кассационный суд;
- заявитель;
- процессуальное положение;
- номер дела;
- решение первой инстанции;
- апелляционное определение;
- какие судебные акты обжалуются;
- существенные нарушения норм права;
- требования заявителя;
- приложения.

## Важно

Кассация не должна быть повторной апелляцией.

LexPilot должен фокусироваться на существенных нарушениях норм материального или процессуального права.

## Минимальный результат

LexPilot готовит кассационную жалобу по `cassation_generic.md`.
